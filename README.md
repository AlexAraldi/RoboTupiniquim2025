# RoboTupiniquim2025

Nossa equipe foi procurada para desenvolver um software de comando para movimentar os robôs em marte.
Está missão foi concluída com base nos estudos da Academia do Programador 2025.

## Funcionalidades:
- Primeiro etapa: o operador deve informar o tamanho do Grid de atuação dos Robôs.

- Segunda etapa: o operador informa o ponto de partida da área de movimentação dos Robôs no Grid
- Terceira etapa: o operador deve informar os comandos completo de toda a movimentação dos Robôs pelo Grid.

### Obs:
Os Robôs executaram o movimento completo um por vez.

Deve atentar-se as dicas de preenchimento no menu de input do operador.

### Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

### 1. Clone o repositório.

``` 
https://github.com/AlexAraldi/RoboTupiniquim2025.git
```
### 2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz

```
cd RoboTupiniquim2025
```

### 3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

### 4. Navegue até a pasta do projeto

```
cd RoboTupiniquem2025
```

### 5. Execute o projeto

```
dotnet run
```
